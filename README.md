## Задание

Используя HTML, CSS, TypeScript, React, Redux, Node.js и Webpack необходимо с чистого листа создать сайт, на который будет выведен компонент “Select”, кнопка “Отправить” и сообщение о результатах выполнения запроса на сервер.

## Проделанная работа

1. Создание сборки на webpack для fullstack приложения
2. Создание бэкенда на чистом Node.js
3. Создание кастомного селекта
4. Создание кнопки не используя тег button
5. Создание компонента для вывода сообщения об успехе или ошибки
5. Сделал сайт полностью доступных с клавиатуры
7. Для оптимизации данных в компоненте select применен @tanstack/react-virtual для большого объема данных
